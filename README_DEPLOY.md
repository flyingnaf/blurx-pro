# BlurX Pro - Animated GIF Support

## الجديد
- دعم GIF المتحرك فعلياً باستخدام gifuct-js.
- التمويه/البكسلة/التعتيم يطبق على فريمات GIF.
- تصدير GIF متحرك باسم blurx-animated.gif.

## طريقة الرفع
ارفع هذه الملفات فوق الملفات القديمة في GitHub ثم Commit changes.

## ملاحظات
- أول تشغيل يحتاج إنترنت لتحميل مكتبات CDN.
- GIF الكبير جداً قد يكون بطيئاً، لذلك النسخة تحدد أول 120 فريم وتصدر بعرض أقصى 640px.
