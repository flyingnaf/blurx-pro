# BlurX Pro Mobile v4

## ماذا تم تعديله؟
- تحسين اللمس على الجوال.
- تحسين تصدير GIF المتحرك.
- وضع Mobile GIF يقلل الفريمات والحجم لتجنب تعليق الآيفون.
- فتح الملف في نافذة جديدة على iPhone إذا منع Safari التحميل المباشر.
- تعطيل Service Worker القديم حتى لا يستمر الموقع بتشغيل ملفات قديمة من الكاش.
- إضافة `?v=4` على ملفات CSS و JS لكسر الكاش.

## طريقة الرفع على GitHub Pages
1. فك ضغط الملف.
2. افتح مستودع GitHub.
3. احذف الملفات القديمة أو ارفع هذه الملفات فوقها.
4. اضغط Commit changes.
5. انتظر دقيقة.
6. افتح الرابط:
   https://YOUR_USERNAME.github.io/YOUR_REPO/

## مهم جداً بعد الرفع
افتح الموقع ثم:
- على الكمبيوتر: Ctrl + F5
- على iPhone: Settings > Safari > Clear History and Website Data، أو افتح الرابط في Private Tab
- أو أضف `?v=4` آخر الرابط، مثال:
  https://flyingnaf.github.io/blurx-pro/?v=4

## حدود واقعية
تطبيق الويب لا يستطيع ضمان 100% لكل فيديو/GIF ضخم على كل أجهزة iPhone بسبب قيود ذاكرة Safari.
لأفضل نتيجة:
- GIF أقل من 10MB
- أقل من 5 ثواني
- استخدم وضع "ثابتة للجوال"
- للكمبيوتر استخدم Chrome أو Edge
